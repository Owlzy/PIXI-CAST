# PIXI-CAST
Simple PIXI.js raycast engine

A simple raycast engine using PIXI.js that includes rendering of sprites. 

# Demo
http://www.tinyowlgames.com/owldev/demos/pixi-cast/
W,A,S,D to move around

# Instructions
run npm install 

to run watch script:
webpack --watch 

to build:
webpack
